## Usage for developer

### Project setup

Rename dotenvsample file to .env file.

Please set the following catapult network in .env file.

- NETWORK
- NODE_HOST
- NODE_PORT
- NETWORK_GENERATION_HASH


Please run commands.

```sh
# Install libraries
$ yarn install

# Run in localhost
$ yarn serve
```
